A 2D physics simulation built with **Python + Pygame** that demonstrates the **gravitational slingshot effect** using interactive gameplay.

The user can launch a spacecraft and observe how its trajectory changes based on the gravity of different planets.


---

## 🎮 How to Play

1. Run the program
2. Select a planet:
   - `1` → Earth  
   - `2` → Jupiter  
   - `3` → Pluto  
3. Click and drag to launch a spacecraft
4. Watch gravitational slingshots in action
5. Press `ESC` anytime to return to menu

---

## 🧠 Physics 

The simulation uses Newton’s law of gravitation:

F = (G × m₁ × m₂) / r²
Where:
- `F` = gravitational force  
- `G` = gravitational constant (scaled for gameplay)  
- `m1, m2` = masses  
- `r` = distance between objects  

The spacecraft updates velocity using acceleration derived from this force.
